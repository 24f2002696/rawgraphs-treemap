# Market Research Analytics - Treemap Visualization

**Email:** 24f2002696@ds.study.iitm.ac.in

This repository contains a Treemap visualization created using RAWGraphs to analyze market share distribution across product categories and subcategories.

## Visualization
The treemap shows hierarchical market data with:
- Primary categories (Electronics, Clothing, Home & Garden, Sports, Books, Toys)
- Subcategories within each category
- Market values represented by rectangle sizes

## Tools Used
- RAWGraphs (https://rawgraphs.io/)
- Data visualization technique: Treemap
